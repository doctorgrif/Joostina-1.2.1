Подключение плагина:
mosCommonHTML::loadJqueryPlugins('dPassword');
Если уже имеется вызов плагинов jQuery, то можно его просто дополнить строкой вызова именно данного плагина.
В теле шаблона перед </body> добавить либо после вызова целевой textarea :
<script type="text/javascript">jQuery(document).ready( function() {$('input:password').dPassword();});</script>
Использование:
После поля для ввода пароля добавте дополнительный input
<form action="#">
<label for="password">Password:</label>
<input type="password" name="password" id="password" value="" />
<input type="submit" name="submit" value="show password" id="submit" onclick="alert('fieldname: ' + 
$('#password').attr('name') + '\npassword value: '+ jQuery('#password').val()); return false;"/>
</form>
для каждого следующего поля на странице добавляем номер поля в вызов (т.е. password2...password3 и т.д)
<p>and another password field:</p>
<form action="#">
<label for="password2">Password:</label>
<input type="password" name="password2" id="password2" value="" />
<input type="submit" name="submit2" value="show password" id="submit2" onclick="alert('fieldname: ' + 
jQuery('#password2').attr('name') + '\npassword value: '+ jQuery('#password2').val()); return false;"/>
</form>
